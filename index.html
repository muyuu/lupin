jQuery plugin lupin

要素を非表示にして、非表示にした要素がウィンドウの指定した高さまで来ると表示させるプラグイン

Build by Grunt, CoffeeScript.

インストール

jQueryの読み込み

<script src='//code.jquery.com/jquery-1.10.1.min.js'></script>

Pocket.jsの読み込み（なくても問題ない）

<script src='pocket.min.js'></script>

lupin.jsの読み込み

<script src='lupin.min.js'></script>

使い方

セレクタに表示非表示を切り替えたい要素を入れる

デフォルトの表示位置は画面の半分

表示位置の切り替えは offset で指定

表示アニメーションの秒数指定は animationTime で指定

$('.one p').lupin();
$('.two p').lupin({offset:'2/3'});
$('.three p').lupin({animationTime:200});
$('.four p').lupin({offset:'quarter'});

デモ

これがデフォルトの表示位置です。
画面の下から1/3の位置まで来たら表示されます。

このテキストは画面の下から2/3程度に来れば表示されます
オプションで { offset: '2/3' } と指定してください。

表示アニメーションの長さをミリセカンドで指定できます。
オプションで { animationTime: 1000 } とかを指定してください。
このテキストは1000の指定で1秒で表示するテキストです。

表示位置の指定は数値と、特定の文字列でも可能です。
{ offset: 500 }
"half", "quarter", "1/3", "1/4", "2/3", "3/4"
